## 🚀 Se Quiser Contribuir
Por favor, não altere os arquivos da pasta sem aprovação. Use branches com nome padronizado ("design/nome-do-design-que-pretende-adicionar") e para fazer o commit use o padrão com limite de cinquenta caracteres, ("design: verbo no imperativo que descreve o que você fez e o que você fez").

Ex: Você quer adicionar o design da página index, então:
- Nome da branch: design/index 
- Commit: design: adiciona design da pagina index
