## 📌 Índice

* Estrutura da Pasta
* Engenharia de Software e Ferramentas


## 📊  Diagramas UML

Este repositório armazena e organiza a documentação visual e os diagramas arquiteturais do sistema, utilizando a especificação UML (Unified Modeling Language) para mapear os requisitos funcionais e a estrutura orientada a objetos da aplicação.

Os diagramas foram exportados no formato vetorial **SVG**, garantindo alta resolução, leveza e compatibilidade de visualização diretamente em qualquer navegador ou ambiente markdown.

## 💻 Engenharia de Software e Ferramentas

A modelagem de sistemas e o design arquitetural deste projeto foram desenvolvidos utilizando o **Astah**. 

A escolha da ferramenta garantiu:
* **Consistência de Modelos:** Alterações em classes ou métodos se propagam automaticamente por todo o projeto, evitando diagramas dessincronizados.
* **Conformidade UML:** Padrão estrito de sintaxe de acordo com a OMG (Object Management Group).
* **Exportação Vetorial:** Todos os diagramas foram exportados em formato `.svg` a partir do Astah para garantir visualização em alta resolução diretamente no repositório.
